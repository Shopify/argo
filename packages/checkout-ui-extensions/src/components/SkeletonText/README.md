# SkeletonText

SkeletonText is used to provide a low fidelity representation of text content before it appears on the page.

## Props
optional = ?

| Name | Type | Description |
| --- | --- | --- |
| inlineSize? | <code>number</code> | Adjust the inline size when no children are passed.<br /><br />Numbers less than or equal to 1 are treated as percentages and numbers greater than 1 are treated as pixels.<br /><br />Examples: - `500` represents `500px` - `0.5` represents `50%` - `1` represents `100%`  |
